# 🪨 Stone Maze 3D

## Présentation du Projet

Un jeu d'exploration de labyrinthe 3D de type "ball maze runner". Le joueur contrôle une boule en pierre à travers un labyrinthe généré de manière procédurale, avec pour objectif de collecter des pièces d'or. Le jeu est pensé pour être fluide à la fois sur ordinateur (clavier/souris) et sur mobile (tactile/joystick), avec pour objectif à terme d'être exporté en application Android.
